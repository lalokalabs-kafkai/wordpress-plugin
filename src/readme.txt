== Description ==

Kafkai is a machine-learning algorithm that can write articles from scratch. Cutting-edge technology for marketers and SEOs.

This plugin can be used to generate and import articles from [Kafkai](https://kafkai.com/). Learn more in [the Help Article](https://help.kafkai.com/article/268-kafkai-plugin)

== Installation ==

1. Upload the entire `kafkai` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
